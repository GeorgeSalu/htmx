## htmx

Este repositório reúne projetos básicos e exemplos práticos para explorar e aprender a tecnologia HTMX. O objetivo é demonstrar, de forma simples e didática, como utilizar HTMX para criar aplicações web dinâmicas, interativas e com menos JavaScript. Cada projeto aborda um conceito ou funcionalidade específica do HTMX, facilitando o estudo incremental da ferramenta.

Você vai aprender a dominar htmx, desde os fundamentos básicos até a integração com tecnologias avançadas para o desenvolvimento de projetos web completos

- Conceitos fundamentais htmx.
- Tipos de carregamento com htmx.
- Eventos com htmx.
- Requisições com htmx.
- Interceptando e tratando erros.
- Interceptando requisições.
- Projetos práticos com htmx e JavaScript vanila.
- Integração com backend.
- Autenticação com token JWT.
- Integração com TailwindCSS.
- NodeJs.
